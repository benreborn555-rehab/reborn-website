# מרכזי גמילה Reborn — אתר תדמית

אתר תדמית יוקרתי, רספונסיבי ובעברית מלאה (RTL) למרכז גמילה להתמכרויות מסמים, אלכוהול והימורים.
עיצוב נקי בגוונים חמים וטיפוליים עם הדגשי זהב, אנימציות גלילה ותאימות מלאה למובייל ולדסקטופ.

## מבנה הפרויקט

```
reborn-rehab-website/
├── index.html          # עמוד הבית (Hero, אודות, שירותים, נתונים, תהליך, יצירת קשר מהירה)
├── about.html          # אודותינו + מפת Google (רחוב הדר 31, כפר סבא)
├── gallery.html        # גלריית תמונות עם תצוגה מוגדלת (Lightbox)
├── contact.html        # צור קשר — טופס לידים + פרטי התקשרות + מפה
├── css/
│   └── styles.css      # כל העיצוב, ה-RTL והאנימציות
├── js/
│   └── main.js         # תפריט מובייל, אנימציות גלילה, מונים, טופס, Lightbox
└── assets/
    └── images/
        └── logo.png    # << מקמו כאן את הלוגו שלכם (ראו למטה)
```

## הוספת הלוגו (חשוב)

שמרו את קובץ הלוגו הזהב שלכם בנתיב הבא, בשם המדויק `logo.png`:

```
assets/images/logo.png
```

הלוגו כבר מוטמע באתר (`assets/images/logo.png`) ומופיע בתפריט העליון ובתחתית כל העמודים.
כדי להחליף בלוגו אחר — פשוט החליפו את הקובץ באותו שם והנתיב.
מומלץ קובץ PNG עם רקע שקוף, ברוחב של כ-600–1024px.

## צפייה באתר

פשוט פתחו את `index.html` בדפדפן.
להרצה כשרת מקומי (מומלץ, כדי שהמפה והפונטים ייטענו בצורה אופטימלית):

```bash
# Python
python -m http.server 8000
# ואז גלשו אל http://localhost:8000
```

## חיבור הטופס לקבלת פניות

הטופס כרגע מדמה שליחה (הודעת הצלחה ללא שרת). לחיבור אמיתי, ערכו את הפונקציה
`form[data-lead]` בקובץ `js/main.js` ושלחו את הנתונים לשירות שתבחרו
(למשל Formspree, EmailJS, או נקודת קצה משלכם).
